# Créer environnement virtuel

```
python3 -m venv venv
```

### Windows

```
.\venv\Scripts\activate
```

### macOS/Linux

```
source venv/bin/activate
```

# installer les dépendance

```
pip install -r requirements.txt

npm i

```

# lancer le projet :

```
uvicorn api:app --reload

npm start

```

# With Make

```
make dev
```
